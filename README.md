# introduction-to-puppeteer

This is a repository for my "Introduction to Puppeteer talk".

Usage:

```shell
npm i
npm test
```

In order to serve assets from 'assets' folder:

```bash
cd assets
python -m SimpleHTTPServer 3000
```
